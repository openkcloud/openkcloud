# openKcloud (AI반도체 클라우드 서비스 플랫폼)

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

## 개요

이 저장소는 **AI 컴퓨팅 가속기(NPU, PIM 등)** 를 클라우드 서비스로 제공하여 초거대 AI 모델의 학습 및 추론에 필요한 고비용 자원의 활용 장벽을 낮추고, 최적 성능과 운용 비용 최적화를 달성하기 위한 **AI반도체 클라우드 서비스 플랫폼** 기술 개발을 위한 프로젝트입니다.  
본 플랫폼은 **AI반도체 as a Service** 개념을 바탕으로 클라우드의 이점을 극대화하며, 다양한 이기종 장치(CPU, GPU, NPU, PIM 등)의 결합 활용을 지원합니다.

아래 그림은 AI반도체 클라우드 서비스 플랫폼의 개념도입니다.

## 요소 기술

**AI반도체 클라우드 서비스 플랫폼**에 대한 구조도는 다음과 같다.

### AI반도체·GPU 기반 클라우드 서비스 플랫폼 아키텍처 기술

-   NPU, PIM 등 **이기종 AI 반도체와 GPU 연동 아키텍처** 구축
-   K-클라우드 결과물 및 서비스 최적화 기술과의 **인터페이스 중개**

### AI반도체 클라우드 서비스 제공 기술

-   **이기종 AI 반도체 기반 VM/컨테이너 및 Kubernetes 제공**
-   최적 패키징/배포 및 **자원 논리·시간 분할 제공**
-   CPU, GPU, NPU, PIM의 **결합 활용 지원**

### 오케스트레이션 및 성능 최적화 기술

-   **인스턴스 스케줄링 기반 성능 최적화**
-   초거대 AI 모델 특화 **클러스터링 기술**
-   워크로드 격리를 통한 **다중 사용자 환경 지원**

### 모니터링 및 트레이싱 기술

-   특화 **모니터링·로깅·알람 기능 제공**
-   사용량 분석 기반 **비용 예측 기능**
-   **장애 탐지 및 복구 기능** 포함

### 운용 비용 최적화 기술

-   워크로드 기반 **스케줄링 기술**
-   비용 분석을 통한 **인프라 재구성 전략**

### 서비스 관리 및 개방형 API 기술

-   AI 소프트웨어 개발 환경 제공 및 **멀티테넌시 지원**
-   사용자별 IAM 인증, 관리자 포털 및 **개방형 API 제공**

### 초거대 AI 모델 기반 플랫폼 검증 및 최적화

-   성능·호환성 검증 및 **단계별 최적화**
    -   (1차) 범용 서버 + GPU 아키텍처
    -   (3차) 서버 + AI 가속기 검증
    -   (5차) K-클라우드 서버 기반 성능 최적화
-   퍼블릭/프라이빗 클라우드 **양쪽에서 사용 가능**
